JQuery Polo Plugins

jquery.messageBox.js jquery.ajaxPostForm.js jquery.ajaxActions.js

These few jquery plugins are designed to easily work with "ajax actions".
Basically, a set of buttons in a table, that should send POST requests to a server automagically... But also forms with many inputs, textarea or select that you'd like to treat with ajax, in one line of code ;)

jquery.messageBox.js

Jquery utility to display messages in a div

jquery.ajaxPostForm.js

Jquery plugin to send form via ajax POST

jquery.ajaxActions.js

Jquery plugin to handle actions with ajax POST
